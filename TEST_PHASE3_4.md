# Phase 3 & Phase 4 测试流程文档

## 测试环境准备

1. 确保 Ollama 服务已启动
2. 确保模型 `qwen3.5:9b` 和 `nomic-embed-text-v2-moe:latest` 已下载
3. 启动应用: `npm run electron:dev`

## Phase 3: 主动提问引擎测试

### 测试 3.1: 记忆缺口分析

**操作步骤:**
1. 启动应用后，打开浏览器开发者工具 (F12)
2. 在控制台输入并执行:
```javascript
// 获取主动提问引擎实例并分析记忆缺口
const { proactiveEngine } = await import('./src/lib/memory/proactiveEngine.ts')
const gaps = await proactiveEngine.analyzeMemoryGaps()
console.log('记忆缺口分析结果:', gaps)
```

**预期结果:**
- 控制台输出记忆缺口数组，显示缺少的记忆类别
- 例如: 
```javascript
[
  {
    category: 'personal',
    description: '缺少个人信息相关的记忆',
    importance: 10,
    suggestedQuestions: ['你叫什么名字？', '你多大了？']
  },
  {
    category: 'preference',
    description: '缺少喜好相关的记忆',
    importance: 8,
    suggestedQuestions: ['你喜欢吃什么？', '你平时喜欢听什么音乐？']
  }
]
```

---

### 测试 3.2: 智能问题生成

**操作步骤:**
1. 在控制台执行:
```javascript
// 生成一个主动提问
const question = await proactiveEngine.generateQuestion()
console.log('生成的问题:', question)
```

**预期结果:**
- 控制台输出包含问题内容、类别、原因、优先级
- 例如:
```javascript
{
  question: '亲爱的，可以告诉我你叫什么名字吗？我想记住你的名字~',
  category: 'personal',
  reason: '缺少个人信息相关的记忆',
  priority: 10
}
```
- 控制台显示问题改写日志:
```
[ProactiveEngine] 问题改写: "你叫什么名字？" -> "亲爱的，可以告诉我你叫什么名字吗？我想记住你的名字~"
```

### 测试 3.3: 会话计数与冷却机制

**操作步骤:**
1. 在控制台执行:
```javascript

console.log('当前状态:', {
  questionsThisSession: state.questionsThisSession,
```

**预期结果:**
- 显示当前会话已提问次数
- 显示最大提问次数 (默认 3)
- 显示冷却时间 (默认 30 分钟)

### 测试 3.4: 主动提问触发（手动测试）

**操作步骤:**

**预期结果:**
- 控制台输出:
```

```

## 综合测试: 完整流程

### 测试流程: 记忆 → 主动提问 → 管理

**操作步骤:**

1. **清空现有记忆**
   - 右键 → 记忆管理 → 清空所有记忆

2. **触发主动提问**
   ```javascript
   proactiveEngine.setCooldown(0.1)
   proactiveEngine.checkAndAsk()
   ```

3. **回答问题**
   - 双击窗口打开输入框
   - 输入: "我叫张宝楠，今年25岁，喜欢打篮球"
   - 发送

4. **验证记忆提取**
   - 控制台应显示:
   ```
   [MemoryExtractor] 从对话中提取记忆...
   [MemoryExtractor] 提取到 X 条记忆
   [MemoryDB] 添加记忆: 用户的名字叫张宝楠...
   ```

5. **打开记忆管理面板**
   - 右键 → 记忆管理
   - 验证记忆已保存

6. **编辑记忆**
   - 修改某条记忆内容
   - 更改重要性
   - 保存

7. **验证去重**
   - 再次发送 "我叫张宝楠"
   - 控制台应显示:
   ```
   [MemoryDB] 向量相似度: 1.000 vs "用户的名字叫张宝楠..."
   [MemoryDB] 跳过重复记忆: 用户的名字叫张宝楠 相似度: 1.000
   ```

## 常见问题排查

### 问题: 主动提问不触发
**排查:**
1. 检查冷却时间是否过长
2. 检查会话提问次数是否已达上限
3. 检查是否有记忆缺口

### 问题: 记忆面板无法打开
**排查:**
1. 检查控制台是否有错误
2. 检查 MemoryPanel.vue 是否正确导入

### 问题: 记忆不显示
**排查:**
1. 检查数据库文件是否存在
2. 检查 memory.getAll() 是否返回数据
3. 检查控制台错误日志
